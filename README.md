# MyContri Platform

## Overview
**MyContri** is a full-stack platform for managing contributions, on-chain document attestation, and contributor rewards.  
It uses a **Flask REST backend** with a **Vite/React frontend** and integrates **IPFS** for document storage. Users verify identities via email-based KYC before submitting work.

## Features
- Email-based KYC for verified users
- Submit, review, and approve contributions
- Pin supporting documents to IPFS
- Contributor rewards and marketplace tracking
- Admin dashboard for approval & blockchain automation

## Tech Stack
- **Backend:** Python, Flask, SQLAlchemy
- **Frontend:** React, Vite, Tailwind
- **Blockchain:** Smart contracts, IPFS (Pinata)
- **Other:** EmailJS for OTP verification.
